# nodejs-web-reactjs-express-api-ssl-promise-fetch

## Description
A reactjs that uses expressjs api that uses self signed ssl.

Features: 
- component with parameters
- css grid
- fetch
- promise
- cors
- self signed ssl

## Tech stack
- reactjs
- expressjs
- webpack

## Docker stack
- alpine:edge
- node:latest
- bayesimpact/react-base:latest

## To run
`sudo ./install.sh -u`
- [Availble here](http://localhost/)

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
